# InterviewIQ – AI-Powered Mock Interview Platform 🚀

An intelligent, full-stack mock interview platform built using the MERN stack. It simulates real-world interview environments by generating dynamic, role-specific questions and providing instant evaluation.

---

## ✨ Key Features

- **🎯 Dynamic Question Generation**: Uses OpenAI/LLM models to generate role-specific questions tailored to user preferences and extracted resume details via `PDF.js`.
- **📊 Instant AI Feedback**: Evaluates user responses immediately, offering constructive insights on communication skills, technical correctness, and core strengths.
- **💳 Credit-Based Razorpay Integration**: Features a secure payment gateway integration using Razorpay with a fully functional credit system for managing premium interview tokens.
- **🎨 Slick UI Dashboards**: Designed a modern, highly responsive frontend architecture powered by `Redux Toolkit` for robust state management and `Tailwind CSS`.

---

## 🛠️ Tech Stack

**Frontend:**
- React.js (Vite)
- Redux Toolkit (State Management)
- Tailwind CSS

**Backend & Database:**
- Node.js & Express.js
- MongoDB (Database Connectivity)

**Libraries & Services:**
- OpenAI API / LLM Integrations
- PDF.js (For parsing and extracting data from resumes)
- Firebase Authentication (Secure user onboarding)
- Razorpay API (Payment gateway architecture)
